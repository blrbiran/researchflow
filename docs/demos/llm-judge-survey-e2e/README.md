# LLM-as-a-Judge Survey E2E Demo

This demo shows the minimum end-to-end ResearchFlow artifact chain for a **survey / synthesis paper** topic:

> Reliability and validity in LLM-as-a-judge evaluation.

Unlike the technical/system and benchmark demos, this example contributes a taxonomy and cross-study synthesis rather than a new system or dataset.

The literature entries are illustrative seeds for workflow testing, not a verified survey corpus. A real run must retrieve and verify every cited work.
