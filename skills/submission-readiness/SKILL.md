# Submission Readiness

Use this skill for the final gate before submitting or sharing a polished draft.

## When to use

- The user asks if the paper is ready to submit.
- The draft is close to camera-ready or final internal review.
- The user wants a final checklist rather than another broad research pass.

## When not to use

- The paper still has major structural or evidence issues. Use `paper-review` first.
- The artifact surface is still incomplete. Use `artifact-packaging` first.

## Review frame

This is a final gate, not an exploratory critique. The focus is:
- venue fit
- abstract / contribution / result alignment
- obvious claim overreach
- figure and caption readiness
- artifact and appendix completeness
- avoidable reviewer irritation from formatting or surface issues

## Workflow

1. Confirm the target venue or intended audience if known.
2. Check whether any CRITICAL review issues remain unresolved.
3. Review the abstract, contribution claims, and headline results together.
4. Check figure captions, appendix / supplement references, and artifact readiness.
5. Return a go / revise judgment with the smallest blocking fix list.

## Output

Return:
- go / revise recommendation
- blocking issues
- high-value fixes before submission
- final confidence note
